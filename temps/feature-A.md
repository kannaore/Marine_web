# GSAP + R3F 중심 컨셉 계획서 (라이트웨이트 & 임팩트)

## 0) 콘셉트 한 줄
- 바다의 정밀함과 미래감을 3D와 모션으로 각인하는 프리미엄 B2B 쇼케이스

## 1) 경험 목표
- 3초 내 브랜드 메시지 인지
- 30초 내 신뢰 포인트(수치/인증/클라이언트) 3개 인지
- 60초 내 문의/브로셔 CTA 클릭 유도
- 백엔드는 최소화, 정적 콘텐츠 중심

## 2) 시각 방향 (세련 + 눈에 확)
- 타이포: 대형 헤드라인 + 넓은 여백 + 짧고 강한 서브카피
- 컬러: marine-dark 베이스 + ocean/cyan 하이라이트
- 소재: 글래스 패널 + 미묘한 그라데이션/노이즈 배경
- 구성: 풀블리드 이미지/3D, 비대칭 레이아웃, 섹션 리듬감

## 3) 모션 시스템 (GSAP)
- 기본 모드(상시): 12~24px 상승 + opacity, 0.6~0.8s
- 스크롤 모션: 패럴럭스(배경/3D), 텍스트 reveal, 섹션 고정
- 상호작용: 카드 lift, hover glow, 네비 모핑
- 버스트 모드(필요할 때 폭발): Hero 최초 진입/섹션 전환/CTA 트리거에서 2~6초 집중 연출
- 가드레일: 기본은 섹션당 1~2개, 버스트 순간만 3~4개 허용, 한 화면 1회
- 접근성: `prefers-reduced-motion`/저사양에서는 버스트 축소 또는 비활성

## 4) 3D 시스템 (R3F)
- 사용 위치: Hero 1개 + 강조 섹션 1~2개만
- 씬 구성: 저폴리 중심, 메시 3~6개, 드로우콜 150 이하
- 버스트 3D: 카메라 push-in, 오브젝트 속도/왜곡 강화, 짧은 파티클 연출(2~4초) 후 기본 상태 복귀
- 성능 전략:
  - `dpr={[1, 1.5]}` 기본, 모바일은 1로 고정
  - `PerformanceMonitor`로 성능 저하 시 디테일 축소
  - `prefers-reduced-motion`/저사양 디바이스는 2D 대체
  - 텍스처 2K 이하, 투명도/쉐이더 복잡도 최소화
- 연동 방식: GSAP 타임라인으로 카메라/오브젝트 시네마틱 연출

## 5) 페이지 구조 (콘텐츠 최소화)
- Home: Hero(3D) → Capabilities → Proof(Stats/Certs) → Case Studies → CTA
- Services: 요약 + 핵심 분야 3~5개 + 문의
- About/Assets/Performance: 카테고리 랜딩 + 디테일 템플릿
- Contact: 빠른 문의 + 오피스 정보

## 6) 단계식 개발 순서 (컨셉 반영)
1. 메시지/콘텐츠 확정 (1일)
   - 브랜드 메시지, KPI, CTA 문구 확정
   - 산출물: 페이지별 카피표 (한/영)
2. 디자인 토큰 & 레이아웃 잠금 (2일)
   - `globals.css` `@theme` 정리, 섹션 간격 규칙 확정
   - 산출물: 버튼/카드/섹션 가이드
3. Hero 3D 프로토타입 (2~3일)
   - R3F 씬 1개 + 카메라 무빙 + 기본 라이팅
   - 산출물: 3D Hero 초안
4. GSAP 모션 시스템 구축 (2일)
   - 진입/스크롤/상호작용 패턴 템플릿화
   - 산출물: 모션 패턴 가이드 + 예제 섹션
5. 핵심 섹션 3개 완성 (3~5일)
   - Hero, Capabilities, Proof 중심으로 완성도 극대화
   - 산출물: 메인 전개 흐름 완성
6. 나머지 페이지 템플릿 적용 (2~3일)
   - Landing/Detail 공통 템플릿 일괄 적용
   - 산출물: About/Services/Assets 샘플 완성
7. 로컬라이즈/카피 정리 (1~2일)
   - 한글 인코딩 문제 해결 + 문장 다듬기
   - 산출물: i18n 정상 표기
8. 성능/품질 최적화 (1~2일)
   - 이미지 sizes, 3D fallback, 모션 절제
   - 산출물: Lighthouse 80+ 목표
9. 배포 준비 & QA (1일)
   - 메타/OG 점검, 모바일/데스크탑 최종 체크
   - 산출물: 배포 체크리스트 완료

## 7) 산출물 체크리스트
- Hero 3D 씬 + GSAP 타임라인
- 섹션별 모션 패턴 가이드
- 저사양/Reduced Motion 대응 분기
- 모바일/데스크탑 QA 완료
